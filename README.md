## 🚀 Featured Projects

### 📄 Enterprise Document QA — SEC Filing RAG System

A financial document question-answering system built for SEC filings.

**Key features:**

- Hybrid retrieval using **BM25 and Qdrant semantic search**
- Reciprocal Rank Fusion for combining retrieval results
- Cross-encoder re-ranking
- Multi-turn conversation memory
- Query rewriting for follow-up questions
- Semantic caching
- Automated LLM-as-a-Judge evaluation
- FastAPI backend with Server-Sent Events

**Results:**

- Improved overall RAG evaluation score from **0.7333 to 0.7583**
- Improved context precision from **0.3833 to 0.4750**
- Reduced repeated query latency to approximately **0.108 seconds**

**Technologies:**

`Python` `FastAPI` `Qdrant` `Sentence Transformers` `BM25` `Cross-Encoder` `Groq` `Gemini` `BeautifulSoup` `tiktoken` `Pytest`

[View Repository](https://github.com/NamTV2712/Enterprise-Document-QA)
